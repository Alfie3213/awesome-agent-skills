## 详细介绍

### 1. [nexu-io/open-design](https://github.com/nexu-io/open-design)

nexu-io/open-design 是一个本地优先的开源设计工具，定位为 Anthropic Claude Design 的开源替代方案。该项目提供 19 个专业化 Skills 和 71 个品牌级设计系统，支持生成网页、桌面、移动端原型、幻灯片、图片和视频。具备沙箱预览功能，支持 HTML/PDF/PPTX/MP4 导出，可运行于 Claude Code、Codex、Cursor、Gemini、OpenCode、Qwen、Copilot、Hermes、Kimi CLI 等主流 AI 编码工具。

**GitHub链接**：https://github.com/nexu-io/open-design

### 2. [alchaincyf/huashu-design](https://github.com/alchaincyf/huashu-design)

huashu-design 是 Claude Code 的 HTML 原生设计 skill，支持高保真原型、幻灯片和动画制作。该项目融合了 20 个设计哲学、5 维评审标准和 MP4 导出功能，采用 Agent 无关的设计理念，可适配多种 AI 代理工作流。

**GitHub链接**：https://github.com/alchaincyf/huashu-design

### 3. [kyegomez/OpenMythos](https://github.com/kyegomez/OpenMythos)

OpenMythos 是对 Claude Mythos 架构的理论重建项目，基于现有研究文献从第一性原理构建。该项目旨在深入理解 Claude 架构的设计理念和技术实现，为 AI 代理系统的研究提供理论参考。

**GitHub链接**：https://github.com/kyegomez/OpenMythos

### 4. [browser-use/browser-harness](https://github.com/browser-use/browser-harness)

Browser Harness 是一个自愈型浏览器 harness，使 LLMs 能够完成任何任务。该项目通过自动化方式处理网页交互中的各种异常情况，提高 AI 代理在浏览器环境中的稳定性和可靠性。

**GitHub链接**：https://github.com/browser-use/browser-harness

### 5. [h4ckf0r0day/obscura](https://github.com/h4ckf0r0day/obscura)

Obscura 是专为 AI 代理和网页爬虫设计的无头浏览器工具，提供强大的网页自动化和数据抓取能力，支持多种复杂场景下的浏览器控制和数据提取。

**GitHub链接**：https://github.com/h4ckf0r0day/obscura
